## Giới thiệu đề tài
Green Action Management System là hệ thống hỗ trợ quản lý các hoạt động bảo vệ môi trường, giúp kết nối giữa thành viên và tổ chức để xây dựng một cộng đồng xanh hiện đại, minh bạch và tiện lợi.

Hệ thống hỗ trợ quản lý sự kiện môi trường, quỹ đóng góp, check-in tham gia bằng GPS, diễn đàn cộng đồng và chứng nhận điện tử cho người tham gia.

---

## Chức năng cốt lõi
- Quản lý sự kiện và hoạt động môi trường
- Đăng ký và check-in tham gia bằng GPS
- Quản lý quỹ và sao kê minh bạch
- Diễn đàn chia sẻ cộng đồng
- Hệ thống điểm thưởng và chứng nhận điện tử
- Dashboard thống kê và quản lý dữ liệu

---

## Công nghệ sử dụng
- ASP.NET MVC (C#)
- Microsoft SQL Server
- HTML5, CSS3, JavaScript
- Google Maps API & GPS Integration

---

## Mục tiêu
Xây dựng một nền tảng hỗ trợ tổ chức và lan tỏa các hoạt động xanh hiệu quả, minh bạch và thân thiện với giới trẻ.
